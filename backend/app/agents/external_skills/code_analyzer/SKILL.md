# Code Analyzer

## 概述

此Skill用于分析代码质量、识别代码异味、提供改进建议。

## 指令

### 1. 代码结构分析
- 分析代码组织结构
- 识别模块职责
- 评估代码可维护性

### 2. 代码质量检查
- 检查命名规范
- 识别代码异味
- 检测潜在bug

### 3. 改进建议
- 提供重构建议
- 建议设计模式
- 推荐最佳实践

### 4. 文档生成
- 生成代码文档
- 创建架构图
- 编写使用说明

## 最佳实践

- 遵循SOLID原则
- 使用类型注解
- 编写清晰的文档字符串
- 保持代码简洁和可读性
